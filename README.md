# Citibikes_Tableau

Using Tableau, we are able to break down the different trips of bikers for NYC. We break down the data first using pandas to calculate trip durations for each person. After using Pandas, we are able to upload the CSV to Tableau and break down the data using the preprogrammed graphing systems.
